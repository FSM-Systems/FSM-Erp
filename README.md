# FSM-Erp
A simple management tool for small companies. 

Containes modules for:
- Maintenance
- Workshop
- Stores
- Fuel
- HR

Modules can be added and customized. Refere to the [wiki] (https://github.com/FSM-Systems/FSM-Erp/wiki) for more information.

### Components used:
- Express.js
- jQuery
- Boostrap
